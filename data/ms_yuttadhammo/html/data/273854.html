10.1.6 Piyaṅkarasutta
Yakkhasaṁyutta
Indakavagga
Piyaṅkarasutta
240.
1479Ekaṁ samayaṁ āyasmā anuruddho sāvatthiyaṁ viharati jetavane anāthapiṇḍikassa ārāme. Tena kho pana samayena āyasmā anuruddho rattiyā paccūsasamayaṁ paccuṭṭhāya dhammapadāni bhāsati. Atha kho piyaṅkaramātā yakkhinī puttakaṁ evaṁ tosesi— 
1480
“Mā saddaṁ kari piyaṅkara,
Bhikkhu dhammapadāni bhāsati;
Api ca dhammapadaṁ vijāniya,
Paṭipajjema hitāya no siyā.
1481
Pāṇesu ca saṁyamāmase,
Sampajānamusā na bhaṇāmase;
Sikkhema susīlyamattano,
Api muccema pisācayoniyā”ti.
