1.3.5.1 Ajjhattasutta
Khandhasaṁyutta
Uparipaṇṇāsaka
Diṭṭhivagga
Ajjhattasutta
150.
792Sāvatthinidānaṁ. “Kismiṁ nu kho, bhikkhave, sati, kiṁ upādāya uppajjati ajjhattaṁ sukhadukkhan”ti? Bhagavaṁmūlakā no, bhante, dhammā…pe… “rūpe kho, bhikkhave, sati rūpaṁ upādāya uppajjati ajjhattaṁ sukhadukkhaṁ. Vedanāya sati…pe… saññāya sati… saṅkhāresu sati… viññāṇe sati viññāṇaṁ upādāya uppajjati ajjhattaṁ sukhadukkhaṁ. Taṁ kiṁ maññatha, bhikkhave, rūpaṁ niccaṁ vā aniccaṁ vā”ti? “Aniccaṁ, bhante”. “Yaṁ panāniccaṁ dukkhaṁ vā taṁ sukhaṁ vā”ti? “Dukkhaṁ, bhante”. “Yaṁ panāniccaṁ dukkhaṁ vipariṇāmadhammaṁ, api nu taṁ anupādāya uppajjeyya ajjhattaṁ sukhadukkhan”ti? “No hetaṁ, bhante”. “Vedanā…pe… saññā… saṅkhārā… viññāṇaṁ niccaṁ vā aniccaṁ vā”ti? “Aniccaṁ, bhante”. “Yaṁ panāniccaṁ dukkhaṁ vā taṁ sukhaṁ vā”ti? “Dukkhaṁ, bhante”. “Yaṁ panāniccaṁ dukkhaṁ vipariṇāmadhammaṁ, api nu taṁ anupādāya uppajjeyya ajjhattaṁ sukhadukkhan”ti? “No hetaṁ, bhante”. “Evaṁ passaṁ…pe… nāparaṁ itthattāyāti pajānātī”ti.
793Paṭhamaṁ.
