1.4.6 Dutiyamittasutta
Paṭhamapaṇṇāsaka
Devatāvagga
Dutiyamittasutta
37.
197“Sattahi, bhikkhave, dhammehi samannāgato bhikkhu mitto sevitabbo bhajitabbo payirupāsitabbo api panujjamānenapi. Katamehi sattahi? Piyo ca hoti manāpo ca garu ca bhāvanīyo ca vattā ca vacanakkhamo ca gambhīrañca kathaṁ kattā hoti, no ca aṭṭhāne niyojeti. Imehi kho, bhikkhave, sattahi dhammehi samannāgato bhikkhu mitto sevitabbo bhajitabbo payirupāsitabbo api panujjamānenapīti.
198
Piyo garu bhāvanīyo,
vattā ca vacanakkhamo;
Gambhīrañca kathaṁ kattā,
no caṭṭhāne niyojako.
199
Yamhi etāni ṭhānāni,
saṁvijjantīdha puggale;
So mitto mittakāmena,
atthakāmānukampato;
Api nāsiyamānena,
bhajitabbo tathāvidho”ti.
200Chaṭṭhaṁ.
