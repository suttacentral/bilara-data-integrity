Mahāvāra

Anulomapuggala
198.
1359»  Yo kāmarāgānusayaṁ parijānāti so paṭighānusayaṁ parijānātīti? Āmantā.
1360«  Yo vā pana paṭighānusayaṁ parijānāti so kāmarāgānusayaṁ parijānātīti? Āmantā.
1361»  Yo kāmarāgānusayaṁ parijānāti so mānānusayaṁ parijānātīti?
1362Tadekaṭṭhaṁ parijānāti.
1363«  Yo vā pana mānānusayaṁ parijānāti so kāmarāgānusayaṁ parijānātīti? No.
1364»  Yo kāmarāgānusayaṁ parijānāti so diṭṭhānusayaṁ…pe…  vicikicchānusayaṁ parijānātīti? No.
1365«  Yo vā pana vicikicchānusayaṁ parijānāti so kāmarāgānusayaṁ parijānātīti?
1366Tadekaṭṭhaṁ parijānāti.
1367»  Yo kāmarāgānusayaṁ parijānāti so bhavarāgānusayaṁ…pe…  avijjānusayaṁ parijānātīti?
1368Tadekaṭṭhaṁ parijānāti.
1369«  Yo vā pana avijjānusayaṁ parijānāti so kāmarāgānusayaṁ parijānātīti? No.
