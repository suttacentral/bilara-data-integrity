4.1.10 Seyyajātaka
Catukkanipāta
Kāliṅgavagga
Seyyajātaka
37.
928
“Sasamuddapariyāyaṁ,
mahiṁ sāgarakuṇḍalaṁ;
Na icche saha nindāya,
evaṁ seyya vijānahi.
38.
929
Dhiratthu taṁ yasalābhaṁ,
dhanalābhañca brāhmaṇa;
Yā vutti vinipātena,
adhammacaraṇena vā.
39.
930
Api ce pattamādāya,
anagāro paribbaje;
Sāyeva jīvikā seyyo,
yā cādhammena esanā.
40.
931
Api ce pattamādāya,
anagāro paribbaje;
Aññaṁ ahiṁsayaṁ loke,
api rajjena taṁ varan”ti.
932Seyyajātakaṁ dasamaṁ.
Kāliṅgavaggo paṭhamo.
933Tassuddānaṁ
934
Vivarañca adeyya samiddhavaraṁ,
Atha daddara pāpamahātiraho;
Atha koli palāsavarañca kara,
Carimaṁ sasamuddavarena dasāti.
