2.3.4 Paṭilomapuggala
Mahāvāra
Pajahanavāra
Paṭilomapuggala
165.
1110»  Yo kāmarāgānusayaṁ nappajahati so paṭighānusayaṁ nappajahatīti? Āmantā.
1111«  Yo vā pana paṭighānusayaṁ nappajahati so kāmarāgānusayaṁ nappajahatīti? Āmantā.
1112»  Yo kāmarāgānusayaṁ nappajahati so mānānusayaṁ nappajahatīti?
1113Aggamaggasamaṅgī kāmarāgānusayaṁ nappajahati, no ca so mānānusayaṁ nappajahati. Dvinnaṁ maggasamaṅgīnaṁ ṭhapetvā avasesā puggalā kāmarāgānusayañca nappajahanti mānānusayañca nappajahanti.

1115Anāgāmimaggasamaṅgī mānānusayaṁ nappajahati, no ca so kāmarāgānusayaṁ nappajahati. Dvinnaṁ maggasamaṅgīnaṁ ṭhapetvā avasesā puggalā mānānusayañca nappajahanti kāmarāgānusayañca nappajahanti.
