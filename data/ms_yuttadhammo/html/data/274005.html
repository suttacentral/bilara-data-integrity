3.4.6 Abhinandasutta
Dhātusaṁyutta
Catutthavagga
Abhinandasutta
119.
809Sāvatthiyaṁ viharati. “Yo, bhikkhave, pathavīdhātuṁ abhinandati, dukkhaṁ so abhinandati. Yo dukkhaṁ abhinandati, aparimutto so dukkhasmāti vadāmi. Yo āpodhātuṁ abhinandati…pe… yo tejodhātuṁ… yo vāyodhātuṁ abhinandati, dukkhaṁ so abhinandati. Yo dukkhaṁ abhinandati, aparimutto so dukkhasmāti vadāmi.
810Yo ca kho, bhikkhave, pathavīdhātuṁ nābhinandati, dukkhaṁ so nābhinandati. Yo dukkhaṁ nābhinandati, parimutto so dukkhasmāti vadāmi. Yo āpodhātuṁ…pe… yo tejodhātuṁ… yo vāyodhātuṁ nābhinandati, dukkhaṁ so nābhinandati. Yo dukkhaṁ nābhinandati, parimutto so dukkhasmāti vadāmī”ti.
811Chaṭṭhaṁ.
